# 追剧日历管家 (Drama Calendar Manager)

## 技能简介

追剧日历管家是一个 AI 驱动的智能追剧管理工具。它帮你统一管理所有在追的剧集，自动追踪更新进度、预测完结时间，并在剧荒时智能推荐下一部好剧。

**AI 核心价值：没有 AI，这就是一个普通的 Excel 表格；有 AI，它变成了一个"懂你的追剧管家"。**

## 核心功能

### 📺 功能1：追剧录入（智能解析）

用户用自然语言描述追剧进度，AI 自动提取关键信息。

**触发示例：**
- "我在看《黑暗荣耀》，更新到第8集"
- "最近在追《繁花》，看到第12集了"
- "《庆余年2》看到第15集了"

**AI 提取字段：**
- 剧名
- 当前集数
- 录入日期（自动填入当天）

**输出格式：**
✅ 已记录：
📺 剧名：黑暗荣耀
📌 当前集数：第 8 集
📅 记录时间：2026-07-12

---

### 📅 功能2：日程生成（聚合推送）

根据各剧的播出规律，自动生成本周追剧日程表。

**触发示例：**
- "本周追剧日历"
- "这周看什么"
- "给我追剧日程"

**输出格式：**
📅 本周追剧日历（2026-07-13 ~ 2026-07-19）

┌─────────┬──────────────────────────────────────┐
│ 周一 │ 《繁花》更新第 13 集 │
│ 周二 │ （无更新，休息一下） │
│ 周三 │ 《庆余年2》更新第 22 集 │
│ 周四 │ （无更新，休息一下） │
│ 周五 │ 《黑暗荣耀》更新第 9-10 集 ⭐ 双更 │
│ 周六 │ （无更新，休息一下） │
│ 周日 │ 《繁花》更新第 14 集 │
└─────────┴──────────────────────────────────────┘

📊 本周共 4 集更新，加油追剧！

---

### 🔮 功能3：进度预测（完结预测）

根据剧集总集数和播出频率，预测完结日期。

**触发示例：**
- "《黑暗荣耀》什么时候完结"
- "预测一下《繁花》的完结时间"
- "距离《庆余年2》完结还有多久"

**AI 计算逻辑：**
- 总集数 - 当前集数 = 剩余集数
- 剩余集数 ÷ 每周更新集数 = 剩余周数
- 当前日期 + 剩余周数 = 预计完结日期

**输出格式：**
🔮 《黑暗荣耀》完结预测

📺 总集数：16 集
📌 当前进度：第 8 集（已追 50%）
📅 播出规律：每周五更新 2 集
⏳ 剩余集数：8 集
📆 预计完结日期：2026-08-09（距今约 4 周）
⚡ 距离完结还有：8 集 / 4 周

加油！胜利在望！🎉

---

### 💡 功能4：智能推荐（剧荒模式）

根据你的追剧历史和偏好，AI 智能推荐 3-5 部你可能喜欢的剧。

**触发示例：**
- "剧荒了，推荐几部剧"
- "有什么好看的推荐一下"
- "我该看什么"

**AI 分析维度：**
- 你追过的剧的题材偏好（悬疑、爱情、古装等）
- 你给过好评的剧的共同特征
- 当前热门剧集匹配度

**输出格式：**
💡 根据你的追剧历史，为你推荐：

🎬 《狂飙》
📌 推荐理由：因为你喜欢《黑暗荣耀》的悬疑犯罪风格
⭐ 豆瓣评分：8.5

🎬 《三体》
📌 推荐理由：因为你喜欢《繁花》的高质量制作水准
⭐ 豆瓣评分：8.7

🎬 《漫长的季节》
📌 推荐理由：悬疑+年代剧，你的两大偏好完美结合
⭐ 豆瓣评分：9.4

👉 回复"想看《剧名》"加入追剧清单

---

### 📝 功能5：前情提要生成（一键回顾）

输入剧名和集数，AI 自动生成该集的内容摘要，帮你回忆剧情。

**触发示例：**
- "《黑暗荣耀》第7集讲了什么"
- "帮我回忆一下《繁花》第10集"
- "前情提要：《庆余年2》第20集"

**AI 能力：**
- 根据剧名+集数检索或生成剧情摘要
- 结构化输出：主要事件 + 关键台词 + 伏笔提示

**输出格式：**
📝 《黑暗荣耀》第 7 集 前情提要

📌 主要事件：

文东恩发现河道英的真实身份

朴妍珍开始调查文东恩的背景

朱如炡向文东恩表白

💬 关键台词：
"你以为你赢了？游戏才刚刚开始。"

🔮 本集伏笔：

河道英的过去将在下一集揭晓

新的复仇计划正在酝酿

👉 继续看第 8 集吧！

## 使用方式

在 Hermes Agent 中，直接向 AI 发送自然语言指令即可触发对应功能：

| 你想做的事 | 说一句话 |
|------------|----------|
| 记录追剧进度 | "我在看《黑暗荣耀》，更新到第8集" |
| 查看本周日程 | "本周追剧日历" |
| 预测完结时间 | "《繁花》什么时候完结" |
| 求推荐 | "剧荒了，推荐几部" |
| 回忆剧情 | "《庆余年2》第20集讲了什么" |

## 技术实现

- **自然语言解析**：调用 LLM API 进行语义理解和信息提取
- **日程生成**：基于播出规律配置，计算并格式化输出
- **完结预测**：基于当前进度和播出频率进行推算
- **智能推荐**：调用 LLM API 进行兴趣画像匹配
- **前情提要**：调用 LLM API 生成剧情摘要
